## 🧠 Core Logic Example

```javascript
// Dynamic API aggregation system
async function fetchAnimeData(sources) {
  const results = await Promise.all(
    sources.map(src => fetch(src).then(res => res.json()))
  );

  return results.reduce((acc, data) => {
    return [...acc, ...data.entries];
  }, []);
}
````
