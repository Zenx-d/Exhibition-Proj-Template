## 🧪 SECTION 4: THE NEW MECHANIC

The **Zen Widget** system allows you to build entire websites in their own folders and embed them instantly using a single line of HTML:

```html
<div data-zen-widget="category/widget-name" data-height="400"></div>
```

**Benefits:**
- **Isolation**: Each widget has its own CSS and JS context.
- **Performance**: Widgets lazy-load only when they enter the viewport.
- **Stability**: Zero interference with the Next.js hydration cycle.
